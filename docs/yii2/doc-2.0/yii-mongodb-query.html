Class yii\mongodb\Query

All Classes | Properties | Methods
Inheritanceyii\mongodb\Query » yii\base\Component » yii\base\Object

Available since version2.0
Source Code https://github.com/yiisoft/yii2/blob/master/extensions/mongodb/Query.php

Query represents Mongo "find" operation.

Query provides a set of methods to facilitate the specification of "find" command. These methods can be chained together.

For example,

$query = new Query;
// compose the query
$query->select(['name', 'status'])
    ->from('customer')
    ->limit(10);
// execute the query
$rows = $query->all();
