Class yii\jui\Dialog

All Classes | Properties | Methods
Inheritanceyii\jui\Dialog » yii\jui\Widget » yii\base\Widget » yii\base\Component » yii\base\Object
Implementsyii\base\ViewContextInterface
Available since version2.0
Source Code https://github.com/yiisoft/yii2/blob/master/extensions/jui/Dialog.php

Dialog renders an dialog jQuery UI widget.

For example:

Dialog::begin([
    'clientOptions' => [
        'modal' => true,
    ],
]);

echo 'Dialog contents here...';

Dialog::end();

See also http://api.jqueryui.com/dialog/.
