session_abort

(PHP 5 >= 5.6.0)

session_abort — Discard session array changes and finish session

说明

bool session_abort ( void )

session_abort() finishes session without saving data. Thus original values in session data is kept.

返回值

This function returns TRUE if a session was successfully reinitialized, otherwise FALSE.

参见

  • $_SESSION
  • The session.auto_start configuration directive
  • session_start() - 启动新会话或者重用现有会话
  • session_reset() - Re-initialize session array with original values
  • session_commit() - session_write_close 的别名
