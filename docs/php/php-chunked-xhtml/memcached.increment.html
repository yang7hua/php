Memcached::increment

(PECL memcached >= 0.1.0)

Memcached::increment — 增加数值元素的值

说明

public int Memcached::increment ( string $key [, int $offset = 1 ] )

Memcached::increment()将一个数值元素增加参数offset指定的大小。 如果元素的值不是数值类型，将其作为0处理。如果元素不存在Memcached::increment()失败。

参数

key

要增加值的元素的key。

offset

要将元素的值增加的大小。

返回值

成功时返回元素的新值 或者在失败时返回 FALSE。 如果key不存在Memcached::getResultCode()方法返回Memcached::RES_NOTFOUND。

范例

Example #1 Memcached::increment() 示例

<?php
$m = new Memcached();
$m->addServer('localhost', 11211);

$m->set('counter', 0);
$m->increment('counter');
$m->increment('counter', 10);
var_dump($m->get('counter'));

$m->set('key', 'abc');
$m->increment('counter');
?>

以上例程会输出：

int(11)
string(1) "1"
