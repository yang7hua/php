PHP5.6正在开发当中

Warning

PHP 5.6正在测试当中，本节中提到的特点和变化可能会在5.6最终版本发布之前有所改变。如果你要 部署一个稳定的5.6项目，请在此之前仔细核对本迁移指南。

php5.6中大多数的改善对旧项目没有影响。有一些 不兼容处 和 新功能 需要 注意下，并且在正式布署升级后的项目前应该测试您的代码。

参见 PHP 版本迁移指南 5.0.x, 5.1.x, 5.2.x, 5.3.x, 5.4.x and 5.5.x.
