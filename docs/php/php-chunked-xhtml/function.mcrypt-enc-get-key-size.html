mcrypt_enc_get_key_size

(PHP 4 >= 4.0.2, PHP 5)

mcrypt_enc_get_key_size — 返回打开的模式所能支持的最长密钥

说明

int mcrypt_enc_get_key_size ( resource $td )

返回打开的模式所能支持的最长密钥，以字节为单位。

参数

td

加密描述符。

返回值

返回打开的模式所能支持的最长密钥，以字节为单位。
