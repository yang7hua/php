Memcached::getResultMessage

(PECL memcached >= 1.0.0)

Memcached::getResultMessage — 返回最后一次操作的结果描述消息

说明

public string Memcached::getResultMessage ( void )

Memcached::getResultMessage()返回一个字符串来描述最后一次Memcached方法执行的结果。

参数

此函数没有参数。

返回值

最后一次Memcached操作结果的描述消息。

范例

Example #1 Memcached::getResultMessage() 示例

<?php
$m = new Memcached();
$m->addServer('localhost', 11211);

$m->add('foo', 'bar'); // first time should succeed
$m->add('foo', 'bar');
echo $m->getResultMessage(),"\n";
?>

以上例程会输出：

NOT STORED
