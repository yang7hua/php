Worker::isWorking

(PECL pthreads >= 2.0.0)

Worker::isWorking — 状态检测

说明

public boolean Worker::isWorking ( void )

Worker 对象是否正在执行栈中对象

参数

此函数没有参数。

返回值

布尔值，表示 Worker 对象是否在执行栈中对象

范例

Example #1 检测 Worker 对象状态

<?php
$my = new Worker();
$my->start();
/* ... */
if ($my->isWorking()) {
    /* ... the Worker is busy executing another object */
}
?>
