counter_create

counter_create — 创建一个包含单个数值的计数器。

说明

resource counter_create ( string $name [, integer $initial_value [, integer $flags ]] )

创建一个包含单个数值的计数器。

参数

name
新的计数器名称。
initial_value
计数器的初值值. 默认为零(0)。
flags
新计数器的标志, 从形如 COUNTER_FLAG_* 的常量中选择。

返回值

返回计数器资源。
