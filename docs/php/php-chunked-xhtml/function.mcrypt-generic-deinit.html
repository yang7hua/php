mcrypt_generic_deinit

(PHP 4 >= 4.0.7, PHP 5)

mcrypt_generic_deinit — 对加密模块进行清理工作

说明

bool mcrypt_generic_deinit ( resource $td )

本函数终止由加密描述符（td）指定的加密模块， 它会清理缓冲区，但是并不关闭模块。 要想关闭加密模块， 你需要自行调用 mcrypt_module_close() 函数。 （但是 PHP 会在脚本末尾为你关闭已打开的加密模块）

参数

td

加密描述符。

返回值

成功时返回 TRUE， 或者在失败时返回 FALSE。

参见

  • mcrypt_module_open() - 打开算法和模式对应的模块
  • mcrypt_generic_init() - 初始化加密所需的缓冲区
