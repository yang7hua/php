Imagick::cropThumbnailImage

(PECL imagick 2.0.0)

Imagick::cropThumbnailImage — 裁剪生成缩略图

说明

bool Imagick::cropThumbnailImage ( int $width , int $height )

生成一个指定大小的缩略图，首先放大或缩小图片，然后裁剪出一个指定的区域从图片中间。

参数

width

缩略图的宽度

height

缩略图的高度

返回值

成功时返回 TRUE。

错误／异常

错误时抛出 ImagickException。
