MongoRegex 类

(PECL mongo >=0.8.1)

简介

这个类用于创建一个正则表达式。 通常这些表达式用于查询数据库中匹配的字符串。不常用的是，它们可以保存到数据库并用于检索。

正则表达式由四部分组成。 首先 / 作为起始的分隔符，然后是 pattern、另一个 / 以及最后包含 flag 的字符串。

Example #1 正则表达式

/pattern/flags

Mongo 能够识别六种正则表达式标记（flag）：

  • i：大小写不敏感

  • m：多行

  • x：能够包含注释

  • l：语言环境

  • s：dotall，"." 匹配任何字符，包括换行符。

  • u：匹配 Unicode
